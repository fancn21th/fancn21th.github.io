# 布局

## 业务需求

- 大屏上的信息卡 需要能够根据权限 动态的显示或者隐藏
- 大屏上的信息卡 可能需要支持用户自定义 :rocket:
- 大屏的 Grid 布局设计 纵向为 16 列

## 技术方案

绝佳的技术方案 应该是 <Product>gridster.js</Product> , 它能满足以上所有业务需求.
<Product>vue3-grid-layout</Product> 是开源社区提供的 Vue3 版本.
