# VsCode

## Key Map for 不同操作系统

| Mac       | Windows |
| --------- | ------- |
| Command ⌘ | Control |
| Option ⌥  | Alt     |
| Shift ⇧   | Shift   |
| Left <-   | Left    |
| Right ->  | Right   |

## Workflow in editor

### Line

- 移动光标到 `行末尾`

  - ⌘ + ->

- 移动光标 单词步进

  - ⌥ + ->

- 选择整行

  - ⌘ + ⇧ + ->

- 选择单词

  - ⌥+ ⇧ + ->

- 删除当前行
  - ⌘ + ⇧ + K
