# React DnD Tutorial

## 目的

它的作者是 大名鼎鼎的 `Dan Abramov` （React 核心成员, Redux 作者）。

本次教程分 上下两篇

上篇我我会基于 `Dan` 精彩的原版教程 用中文讲解一遍。

下篇我们蹭热打铁 结合一个真实的应用场景，来实现以拖拽的交互方式来改变表单布局。
